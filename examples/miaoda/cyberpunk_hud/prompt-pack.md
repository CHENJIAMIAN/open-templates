# Cyberpunk HUD Prompt Pack

Use this pack to generate interface concepts with a high-contrast cyber terminal feel.

## Core Prompt

Design a cyberpunk HUD interface with a black background, neon green accents, CRT scanlines, monospace typography, hard-edged panels, and modular data readouts. The layout should feel like a command terminal fused with a futuristic control room. Use glowing borders, square badges, bracketed labels, and restrained grayscale imagery.

## Prompt Variants

### Landing Page

Create a product landing page for a futuristic AI operations platform. Use a full-width hero, metric cards, command-style navigation, and a dense grid of panels showing status, alerts, and activity.

### Dashboard

Create a control dashboard for a real-time operations console. Prioritize status visibility, data density, and crisp hierarchy. Show KPI cards, a line chart, a log stream, and a compact command panel.

### Profile

Create a user profile page for a sci-fi game or simulation platform. Use HUD framing, neon avatars, achievement chips, and a strong terminal aesthetic.

### Data Panel

Create a data visualization panel for telemetry monitoring. Use hard borders, monochrome charts, neon highlights, and a layout that feels built for operators.

## Style Constraints

- Black base only, with neon green as the primary signal color.
- No rounded corners except where a control surface needs it.
- Use uppercase labels, wide tracking, and monospace body copy.
- Keep imagery grayscale unless a neon accent is necessary.
- Prefer stacked panels, overlays, and scanline texture over soft shadows.

## Do Not

- Do not introduce pastel accents.
- Do not use glassmorphism or soft blur effects.
- Do not make the layout feel like a generic SaaS dashboard.
- Do not add playful rounded widgets or decorative gradients that weaken the terminal feel.
