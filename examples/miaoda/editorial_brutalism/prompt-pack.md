# Editorial Brutalism Prompt Pack

Use this pack to generate editorial interfaces with hard contrast, oversized typography, and a print-inspired structure.

## Core Prompt

Design an editorial brutalist interface with a paper-like off-white background, heavy black type, hard-edge shadows, oversized display headings, and disciplined grid structure. The page should feel like a magazine layout crossed with a poster system. Use square corners, boxed content, and minimal color.

## Prompt Variants

### Landing Page

Create a product landing page for a premium design tool. Use dramatic typography, large spacing shifts, strong section labels, and a layout that feels printed rather than soft.

### Editorial Dashboard

Create a dashboard that looks like a printed editorial spread. Pair bold KPI blocks with data tables, dense but controlled grids, and strong black dividers.

### Feature Story

Create a story page for a creative company. Combine headline-heavy typography, pull quotes, and boxed media blocks with a magazine-like rhythm.

## Style Constraints

- Use black, white, and paper tones as the base palette.
- Keep corners square and shadows hard.
- Use oversized headings and extreme scale contrast.
- Preserve generous white space instead of filling every gap.
- Add small accent lines or geometric marks only when needed for rhythm.

## Do Not

- Do not use soft glass panels.
- Do not round the corners.
- Do not overdecorate with gradients or glows.
- Do not reduce the typography to a generic SaaS style.
