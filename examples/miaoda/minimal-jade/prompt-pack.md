# Minimal Jade Prompt Pack

Use this pack to generate calm, business-ready dashboards and product pages with a clean jade accent.

## Core Prompt

Design a minimal jade interface with a pale gray-blue background, crisp white surfaces, subtle shadows, and a jade green primary accent. The page should feel reliable, modern, and data-driven. Use rounded cards, calm typography, light borders, and restrained charts.

## Prompt Variants

### SaaS Dashboard

Create a SaaS dashboard with KPI cards, a ranked list, a chart panel, and a clean call-to-action area. Keep the layout airy and professional.

### Finance Dashboard

Create a finance dashboard with portfolio summary cards, trend charts, and a tidy holdings table. The accent should be used sparingly to highlight positive movement.

### Operations Board

Create an operations board for a small team. Use status cards, a timeline, and a compact task grid. Keep it calm and highly readable.

## Style Constraints

- Use soft gray-blue surfaces and jade as the primary accent.
- Keep shadows light and spacing generous.
- Favor rounded cards and restrained borders.
- Use tabular numbers and clear hierarchy for KPIs.
- Keep the interface practical, not decorative.

## Do Not

- Do not make the palette overly saturated.
- Do not introduce neon or glass effects.
- Do not crowd the layout with too many widgets.
- Do not make the page feel playful or flashy.
