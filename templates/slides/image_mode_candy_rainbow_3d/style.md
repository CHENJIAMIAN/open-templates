# Candy Rainbow 3D 风格描述

**整体色调**：
- 梦幻甜美，高饱和彩虹色系
- 彩虹渐变贯穿所有主要元素：红、橙、黄、绿、青、蓝、紫
- 背景为柔和渐变：薄荷绿到粉色、天蓝到粉紫、彩虹光晕
- 3D糖果质感、游戏化设计、节日庆祝氛围

**风格关键词**：
```
【整体风格】[3D candy wonderland, jelly gummy style, rainbow gradient elements, glossy translucent textures, playful celebration mood, game-like aesthetic, sweet fantasy atmosphere, no flat graphics, no realistic photos, no watercolor, all visual elements must be 3D rendered candy/jelly/glass style]
【配色】(soft gradient backgrounds in mint green to pink or sky blue to lavender, rainbow spectrum for main 3D elements, white for titles, golden accents for milestones, pastel tones throughout)
【字体风格】[标题: 白色圆润粗体带柔和阴影和发光效果, 副标题: 白色或淡色圆体, 数字强调: 3D立体效果带彩色渐变描边, 标签: 胶囊形状背景配白色圆体字]
【背景】(柔和渐变色背景，薄荷绿到粉色或天蓝到粉紫，局部有彩虹光晕，散落彩色小颗粒装饰，整体梦幻朦胧感)
```

**数据展示风格**：
- 毛玻璃效果的半透明数据卡片
- 大号数字用3D立体渐变效果，带彩色描边
- 增长曲线用彩虹色轨道或过山车隐喻
- 时间线用彩虹软糖小路表示
- 里程碑用云朵形状的立体标牌，不同颜色代表不同阶段

**装饰元素**：
- 左上角胶囊形状的标签徽章（渐变背景配白色文字）
- 3D软糖、果冻球、糖果（透明/半透明质感）
- 彩虹色玻璃瓶、盘子等容器
- 棒棒糖、彩虹轨道、过山车
- 透明彩虹气泡
- 彩色小颗粒/糖霜/彩屑散落
- 3D云朵、星星装饰
- 果冻树、糖果树

**场景元素**：
- 彩虹软糖铺成的小路（表示旅程/时间线）
- 糖果堆成的山峰或蛋糕塔（表示成就/目标）
- 彩虹过山车轨道（表示增长曲线）
- 玻璃质感的瓶子装彩虹液体（表示指标容器）
- 云朵形状的指示牌（表示里程碑节点）
- 银色金属托盘上的糖果球（表示数据集合）

**风格一致性约束**：
- 所有视觉元素必须以3D渲染的糖果/果冻/玻璃质感呈现
- 禁止混入扁平图形、写实照片、水彩、手绘等其他风格
- 保持彩虹色系的一致性，所有主要元素都应有彩虹渐变
- 背景必须是柔和的渐变色，不能是纯色或复杂纹理
- 整体氛围保持甜美梦幻、积极向上的庆祝感
