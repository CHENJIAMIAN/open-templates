# Colorful Notebook 风格描述

**整体色调**：
- 活泼明亮，高饱和度多彩
- 米黄牛皮纸背景，温暖亲切
- 标题多色组合，每个字颜色不同
- 课堂笔记感、手账风、趣味学习氛围

**风格关键词**：
```
【整体风格】[hand-drawn notebook style, colorful doodles, washi tape decorations, grid paper background, cute simple illustrations, playful learning aesthetic, sketch notes, bullet journal style]
【配色】(warm beige/kraft paper background with light blue grid lines, multicolor titles in blue/green/orange/pink/purple, colorful sticky note cards in blue/orange/pink/yellow/green, black hand-drawn outlines)
【字体风格】[标题: 多彩手写粗体每字不同色, 副标题: 黑色手写体, 标签: 白色文字彩色底, 正文: 简洁手写风]
【背景】(米黄牛皮纸质感，淡蓝色网格线纹理，边角斜贴彩色胶带条装饰)
```

**内容卡片风格**：
- 圆角矩形卡片，彩色底（蓝/橙/粉/黄/绿）
- 黑色手绘边框勾勒
- 卡片内配简笔画图标
- 卡片之间用手绘箭头连接

**图标与人物风格**：
- 简笔画风格图标（大脑、电脑、书本、灯泡等）
- 可爱简笔画人物，圆润造型，简单表情
- 线条勾勒 + 简单填色

**装饰元素**：
- 斜贴的胶带条（米黄、蓝、粉、绿 washi tape）
- 手绘涂鸦：星星、螺旋、放射线、小箭头、波浪线
- 彩色便签纸、小彩旗、奖章图标
- 网格线背景纹理
- 边角手绘装饰点缀

**风格一致性约束**：
- 所有视觉元素必须以手绘简笔画风格呈现
- 禁止混入写实照片、3D元素、渐变效果等其他风格
