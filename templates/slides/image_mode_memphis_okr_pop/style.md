# Memphis OKR Pop 风格描述

**整体色调**：
- 明亮活泼，高明度高饱和
- 多彩搭配：珊瑚橙、薄荷绿、明黄、粉色为主色调
- 暖米白/奶油色背景
- 90年代孟菲斯波普风、轻松商务感

**风格关键词**：
```
【整体风格】[memphis design, 90s pop art, scattered geometric shapes, corner badge labels, colorful content cards with bold black outlines, playful professional, bold shapes with clean layout, no realistic photos, no watercolor, all visual elements must be flat geometric memphis style]
【配色】(warm cream/off-white background, coral orange for main cards and titles, mint green/lemon yellow/bubblegum pink/sky blue for content cards, bold black outlines for all shapes)
【字体风格】[标题: 超粗黑体带阴影描边有立体感, 副标题: 粉橙渐变色粗体, 正文: 简洁黑体, 角标标签: 粗体配彩色背景黑色边框]
【背景】(暖奶油色/米白背景，四周散落孟菲斯几何装饰，大面积留白，装饰不遮挡内容区)
```

**数据展示风格**：
- 圆角卡片带彩色底和黑色边框
- 大号数字用黑色粗体或彩色
- 简洁柱状图、趋势箭头图标
- 内容分块用不同颜色卡片区分

**装饰元素**：
- 左上角圆角矩形角标标签（彩色背景+黑色边框）
- 几何图形散落边缘：实心三角形、圆点、波浪线、锯齿条纹
- 星形爆炸图案
- 不规则blob流体形状
- 粗箭头指示方向和流程
- 保持大面积留白，装饰集中在四角和边缘

**风格一致性约束**：
- 所有视觉元素必须以孟菲斯扁平几何风格呈现
- 禁止混入写实照片、水彩、3D立体等其他风格
- 所有形状必须有清晰的黑色描边
